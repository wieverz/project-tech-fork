## FilmCrew
FilmCrew is een matching website waar acteurs, filmakers en crewleden elkaar vinden! Of je nu een regisseur bent die een team zoekt, een cameraman die wil mee werken aan een project, een editor, acteur of geluidsman op zoek naar iets nieuws. Op onze site match je gemakkelijk met projecten die passen bij jouw skills en ambities.

Tijdens dit project hebben wij als team vooral gewerkt aan onze ontwikkeling van onze backend. Voor frontend hebben we al een goede basis, maar tijdens dit project konden we ons verdiepen in de server gerelateerde onderdelen. Hierbij hebben we onder andere gewerkt met MongoDB voor het opslaan van data.

### Website lokaal gebruiken
Om onze website lokaal te kunnen bekijken en gebruiken, moet je de volgende stappen volgen.
1. Zorg dat [Node.js](https://nodejs.org/en) en [npm](https://www.npmjs.com/) geïnstalleerd hebt
2. Clone de repository, installeer vervolgens alle packages met (in je terminal): 
> npm i
3. Start daarna de server met (in je terminal):
> npm run dev

Nadat je dit uitvoert in je terminal krijg je een localhost link te zien. Als je deze opent in de browser kun je zo de website lokaal bekijken en gebruiken.

**let op:** de .env file is niet aanwezig. Hierdoor kunnen sommige functionaliteiten mogelijk niet (volledig) werken.
